# # 🚌 Bus Ticket Booking System (Frontend)

[![React](https://img.shields.io/badge/React-20232A?style=for-the-badge&logo=react&logoColor=61DAFB)](https://reactjs.org/)
[![Vercel](https://img.shields.io/badge/Vercel-000000?style=for-the-badge&logo=vercel&logoColor=white)](https://vercel.com/)

A modern and intuitive web application designed to streamline the bus ticket booking process. This project focuses on providing a clean user interface for travelers to search for routes, view bus schedules, and book seats in real-time.

## 🚀 Live Demo
[*View Live Application on Vercel*](https://personal-portfolio-black-pi.vercel.app/) 

## ✨ Key Features
* *Search Engine:* Filter buses based on Source, Destination, and Travel Date.
* *Interactive Seat Map:* Visual seat selection with real-time availability updates.
* *Responsive Design:* Fully optimized for mobile, tablet, and desktop screens.
* *Booking Summary:* Dynamic calculation of total fare and seat details.
* *User Dashboard:* View current and past booking history.

## 🛠️ Tech Stack
* *Library:* React.js (Hooks & Functional Components)
* *Routing:* React Router DOM
* *Styling:* CSS3 / Tailwind CSS
* *State Management:* Context API
* *API Interaction:* Axios for connecting to the Django Backend

## 📦 Getting Started

### Prerequisites
* Node.js (v16 or higher)
* npm or yarn

### Installation
1. *Clone the repository:*
   ```bash
   git clone [https://github.com/himateja27/React-Frontend-BusTicketBooking-System.git](https://github.com/himateja27/React-Frontend-BusTicketBooking-System.git)

   2.Navigate to the project directory:
   cd travels

   3.install dependencies:
   npm install

   4.start the deployment server:
   npm start

   ## 🔌 Backend Integration
This frontend is powered by a custom REST API built with Django.
